Code Organization and Testing

El capítulo se centrará en los temas importantes para mantener tu código mantenible en una aplicación de escala. Vos aprenderá acerca de la organización del código de abrazar las mejores prácticas en la estructuración de tus carpetas y archivos. Otro aspecto que aprenderá es la prueba, lo cual es importante para mantener tu código robusto.

The chapter will focus on important topics to keep your code maintainable in a scaling application. You will learn about code organization to embrace best practices when structuring your folders and files. Another aspect you will learn is testing, which is important to keep your code robust.
