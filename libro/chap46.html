Advanced React Components

El capítulo se centra en la implementación de componentes React avanzadas. Vos aprenderá acerca de los componentes de mayor orden y la forma de aplicarlas. Además va a sumergirse en temas más avanzados en React y poner en práctica las interacciones complejas.

The chapter will focus on the implementation of advanced React components. You will learn about higher order components and how to implement them. In addition you will dive into more advanced topics in React and implement complex interactions.
