Eject

El siguiente paso es el conocimiento y no es necesario para implementar la aplicación a la producción. Aún así, quiero explicar a usted. crear una aplicación React viene con una función para evitar que los proveedores de tecnología. Una dependencia de un proveedor por lo general sucede cuando usted compra en una tecnología pero no hay ninguna vía de escape. En una dependencia de un proveedor que es difícil cambiar la tecnología. En crear una aplicación React que tiene la puerta de escape con la “expulsión”.

The following step and knowledge is not necessary to deploy your application to production. Still, I want to explain it to you. create-react-app comes with one feature to prevent vendor lock-in. A vendor lock-in usually happens when you buy into a technology but there is no escape hatch. In a vendor lock-in it is hard to change the technology. In create-react-app you have the escape hatch with “eject”.

En su package.json se encuentran las secuencias de comandos para iniciar , de prueba y construir tu aplicación. El último guión es de expulsión. Se podía probarlo, pero no hay camino de regreso. Se trata de una operación unidireccional. Una vez que expulse, no se podes volver! . Si vos acaba de comenzar a aprender React, no tiene sentido dejar el medio ambiente conveniente de crear una aplicación React.

In your package.json you will find the scripts to start, test and build your application. The last script is eject. You could try it, but there is no way back. It is a one-way operation. Once you eject, you can’t go back!. If you just started to learn React, it makes no sense to leave the convenient environment of create-react-app.

Si desea ejecutar npm run eject, el comando sería copiar toda la configuración y dependencias a su package.json y una nueva configuración / carpeta. Se podría convertir el proyecto en una instalación personalizada con herramientas que incluye Babel, webpack y ESLint. Después de todo, tenes control total sobre todas estas herramientas.

If you would run npm run eject, the command would copy all the configuration and dependencies to your package.json and a new config/ folder. You would convert the whole project into a custom setup with tooling that includes Babel, Webpack and ESLint. After all, you would have full control over all these tools.

La documentación oficial dice que la aplicación crear una reacción es adecuado para pequeñas y proyectos de tamaño medio. Vos no debes sentirse obligado a utilizar el comando “Expulsar”.

The official documentation says that create-react-app is suitable for small to mid size projects. You shouldn’t feel obligated to use the “eject” command.
